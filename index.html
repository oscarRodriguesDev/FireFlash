FireFlash

Boas vindas!

Bem-vindo à documentação da biblioteca FireFlash! Desenvolvida com o objetivo de simplificar a
utilização dos produtos Firebase, a FireFlash proporciona aos desenvolvedores uma maneira mais intuitiva
e prática de se conectar com o Firebase e usufruir de seus recursos.

Com um processo totalmente open source, disponibilizamos o código-fonte para que os usuários possam ter
acesso e contribuir com o projeto, tornando-o cada vez mais robusto e funcional. Nossa biblioteca oferece
uma solução eficiente para quem busca aumentar a produtividade em seus projetos, sem abrir mão da
qualidade e segurança proporcionadas pelo Firebase.

Este documento tem como objetivo fornecer informações precisas e detalhadas sobre o uso da FireFlash,
permitindo que você aproveite ao máximo todas as funcionalidades disponíveis. Aqui você encontrará desde
instruções básicas de instalação até exemplos práticos de implementação. Esperamos que esta documentação
possa ser útil e esclarecedora para todos os usuários da FireFlash.

Antes de começar...

A biblioteca FireFlash foi projetada para ser utilizada em conjunto com o JavaScript modular.
Essa abordagem modular traz diversos benefícios, como uma estrutura organizada, reutilização de
código e melhor gerenciamento de dependências.

Para aproveitar ao máximo os recursos e funcionalidades da FireFlash, é necessário integrar a,
biblioteca ao seu projeto JavaScript modular.
Isso permitirá uma interação mais eficiente e harmoniosa com os produtos Firebase.

Certifique-se de configurar corretamente o ambiente de desenvolvimento com suporte a JavaScript
modular, seja utilizando ferramentas como o webpack, Rollup ou outra solução equivalente.
Dessa forma, você estará pronto para aproveitar todas as vantagens oferecidas pela biblioteca
FireFlash em seu projeto.

A adoção do JavaScript modular garantirá uma maior organização, manutenibilidade e escalabilidade
ao seu código, além de facilitar a integração com a FireFlash e aprimorar a experiência geral
do desenvolvimento.

Mude a linha onde você importou a biblioteca

Função fireFlash_createAcount

Vamos entender como funciona a função fireFlash_createAcount.

A função fireFlash_createAcount tem 4 parâmetros:

  1. email (string): Contém o endereço de email do usuário a ser criado.
  2. senha (string): Contém a senha do usuário a ser criado.
  3. auth (função): Função nativa do Firebase Authenticator, importada do Firebase,
    usada como parâmetro para autenticação.
  4. native_createLogin_email (função): Função nativa do Firebase Authenticator,
    importada do Firebase, usada como parâmetro para criação de usuário com email e senha.
    No Firebase, o nome dessa função é createUserWithEmailAndPassword.

A função não possui retorno e realiza a criação do usuário no Firebase Auth,
tratando possíveis erros e exibindo mensagens de log no console.

Exemplo de uso:

fireFlash_createAcount('exemplo@email.com', 'senha123',createUserWithEmailAndPassword,getAuth);

Se você tentar executar seu metodo de login agora vai perceber que vamos ober ainda um erro, isso se deve
ao fato de ainda não termos importado o modulo do Authenticator do Firebase, para importar o modulo do Authenticator
você devera adcionar as seguintes linhas no inicio do seu arquivo de configuração: 

import { getAuth,createUserWithEmailAndPassword} from "https://www.gstatic.com/firebasejs/9.17.1/firebase-auth.js";

Função fireFlash_login

Realiza a autenticação de um usuário no Firebase Auth com as credenciais fornecidas (email e senha) e executa a função fornecida como parâmetro caso a autenticação seja bem sucedida.

Parâmetros:

  • email (string) - O endereço de email do usuário a ser autenticado.
  • senha (string) - A senha do usuário a ser autenticado.
  • função (Function) - A função a ser executada após a autenticação bem sucedida do usuário.
  • native_createLogin_email (function) - A função nativa do Firebase utilizada para autenticar o usuário com email e senha.
  • auth (function) - A função nativa do Firebase usada para autenticação de usuários.

Retorno:

Nenhum retorno. A função realiza a autenticação do usuário e trata os possíveis erros, exibindo mensagens de log no console.

Exemplo de uso:

// Exemplo de uso:
fireFlash_login('exemplo@email.com', 'senha123', minhaFuncao);

Antes de finalizar esse topico, é preciso chamar sua atenção para o terceiro paramentro da função
fireFlash_login, isso porque se trata de uma função definida pelo desenvolvedor, utilizador da biblioteca FireFlash
Essa função deve ser criada a fim de informar o usuario que seu login foi bem sucedido, portanto deve se usar
a criatividade e gerar uma interação que consiga dar ao usuario a transparencia de que sua solicitação foi
realizada com sucesso, ou não

fireFlash_logout

Realiza a autenticação de um usuário no Firebase Auth e apaga o cookie de autenticação e o
token de acesso armazenado no navegador.

Depois de chamar essa função, o usuario precisa logar novamente para continuar utilizando os serviços do Firebase
e as demais funcionalidades definidas pelo desenvolvedor

Para utilizar a função fireFlash_logout, é necessario importá-la 

 import {fireFlash_logout} from "https://oscarrodriguesdev.github.io/FireFlash/FireFlash-remote.js";

Entendendo a função fireFlash_logout

Realiza o logout do usuário atualmente autenticado no Firebase Auth e executa a função fornecida como parâmetro caso o logout seja bem sucedido.

Parâmetros:

  • função (Function) - A função a ser executada após o logout bem sucedido do usuário.
  • native_Logout (function) - A função nativa do Firebase utilizada para realizar o logout do usuário.
  • auth (function) - A função nativa do Firebase usada para autenticação de usuários.

Retorno:

Nenhum retorno. A função realiza o logout do usuário e trata os possíveis erros, exibindo mensagens de log no console.

Exemplo de uso:

// Exemplo de uso:
                  fireFLash_Exit(minhaFuncao, signOut(), getAuth);
